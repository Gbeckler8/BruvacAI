# BruvacAI
A LoRA fine-tuned LLM that generates Magic: The Gathering flavor text from card data — built on Llama-3.2-1B-Instruct and trained on 23K+ cards via Scryfall. BruvacAI learns the genre's distinctive voice — terse, evocative prose and quotes drawn from Magic's worldbuilding — and generates new flavor text for any card concept you give it.
